# Social Intents

A helper to open rapidly an social network app or browser window from an url.


## Installation

build.gradle:

```gradle
    allprojects {
	repositories {
	    ...
	    maven { url 'https://jitpack.io' }
	}
    }
```

app/build.gradle dependency:

```gradle
    dependencies {
	    implementation 'com.github.marcandreappel:social_intents:1.0.0'
    }
```
